# SpofunCoach

This repo holds the design work I did for **SpofunCoach**, a fitness/sports coaching app concept — use cases, class diagram, sequence diagrams, activity diagrams, deployment diagram, and the full project report.

There's no source code here. This is purely the analysis and design phase, put together during my final year as a software engineering student. I'm sharing it because when I was working on this project, I struggled to find real, full examples of UML-based analysis online — most of what I found was either too fragmented or way too abstract. So if you're a CS student trying to figure out how to go from "I have an idea for an app" to an actual structured design, maybe this saves you some time.

## What's in here

- Project report (in French — sorry to non-French speakers, might translate parts later)
- Use case diagrams
- Class diagram
- Sequence diagrams
- Activity diagrams
- Deployment diagram
- A few other design notes/artifacts

The report PDF is in the Releases section of this repo, not in the main branch — grab it from there.

## Tools

Drew the UML stuff in draw.io, did UI mockups in Figma, and wrote the report in LaTeX via Overleaf. Nothing fancy, just what I had access to and was comfortable with.

## Why I'm posting this

Mostly for other students. If you're stuck trying to understand how to model a system before writing any code, feel free to dig through this and see how I approached it. It's not perfect — there are definitely things I'd do differently now — but it's a real, complete example rather than a toy one.

Please don't just copy it wholesale for your own assignment though. Use it to understand the reasoning, not as a template to paste in your name and hand in.

If it's useful to you, a star on the repo is appreciated but obviously not required.
